Funciones

Todo lo que necesitas.
Nada más.

Importar por enlace

Pega una URL de YouTube o Spotify y Openfy detecta el tipo automáticamente: playlist, vídeo, álbum o canción. Sin cuentas, sin OAuth.

↳ youtube.com/playlist?list=…

Biblioteca local

Busca, filtra por plataforma, ordena, renombra y elimina playlists. Todo guardado con Room en tu dispositivo.

Reproductor integrado

Cola, shuffle, repeat, mini-player y reproducción en segundo plano con notificación multimedia. Servicio Media3/ExoPlayer.

Audio local

Importa archivos del dispositivo con el selector del sistema y mézclalos en playlists junto a tus referencias importadas.

Export / Import JSON

Exporta toda tu biblioteca — playlists, favoritos, historial y preferencias — a un JSON. Importa en cualquier momento para restaurar.

Sin claves en cliente

La YouTube API key es opcional y nunca se embebe en la APK pública. El proxy de Spotify tampoco expone credenciales en el frontend.

Transparencia legal

Openfy no descarga audio protegido, no evita DRM, no bloquea anuncios de terceros y no reproduce contenido completo de Spotify o YouTube sin permisos válidos.

La app reproduce exclusivamente: archivos locales del usuario, la demo incluida, previews oficiales si un backend seguro las suministra, y streams autorizados. Para el resto, abre la app oficial o el navegador.

La YouTube API key es opcional y se configura en local.properties (fuera del repositorio público). No se distribuye ningún secreto en la APK pública.
